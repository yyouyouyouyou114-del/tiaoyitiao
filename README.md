# 跳一跳小游戏 - 森林主题版本 🌲

一个基于HTML5 Canvas和JavaScript开发的跳一跳小游戏，采用森林主题设计，支持微信小游戏平台和Web浏览器。

## 🎮 游戏特色

### 核心玩法
- **物理引擎**：真实的抛物线跳跃物理效果
- **平台生成**：动态生成不同难度的平台
- **完美着陆**：精确的着陆判定和连击系统
- **分数系统**：基于跳跃距离和连击的计分机制

### 视觉效果
- **森林主题**：绿色森林背景，营造自然氛围
- **动态背景**：飘落的叶子、飞舞的蝴蝶、绽放的花朵
- **气泡效果**：下半部分的气泡动画
- **粒子特效**：跳跃和着陆时的粒子爆炸效果

### 移动端优化
- **响应式设计**：完美适配手机横竖屏
- **触摸控制**：优化的触摸操作体验
- **震动反馈**：跳跃和着陆时的触觉反馈
- **音频系统**：完整的音效和背景音乐

## 🛠️ 安装和运行

### 方法一：本地服务器（推荐）

1. **克隆项目**
   ```bash
   git clone https://github.com/yyouyouyouyou114-del/tiaoyitiao.git
   cd tiaoyitiao
   ```

2. **启动本地服务器**
   ```bash
   # 使用Python
   python -m http.server 8080
   
   # 或使用Node.js
   npx http-server -p 8080
   ```

3. **访问游戏**
   - 桌面端：http://localhost:8080/index.html
   - 移动端：http://localhost:8080/mobile_test.html

### 方法二：直接打开

直接双击 `index.html` 文件在浏览器中打开（部分功能可能受限）。

### 方法三：微信小游戏

1. 下载微信开发者工具
2. 导入项目目录
3. 配置 `game.json` 文件
4. 编译并预览

## 🎯 游戏操作

### 桌面端
- **鼠标点击**：开始蓄力
- **释放鼠标**：跳跃
- **蓄力时间**：影响跳跃距离

### 移动端
- **触摸屏幕**：开始蓄力
- **释放触摸**：跳跃
- **长按时间**：影响跳跃距离

### 游戏技巧
- 观察平台距离，合理蓄力
- 追求完美着陆获得连击奖励
- 注意弹簧平台的弹跳效果
- 保持连击提高分数

## 🎨 自定义配置

### 游戏参数调整
在 `js/jump_game.js` 中的 `CONFIG` 对象可以调整：
- `GRAVITY`：重力大小
- `JUMP_FORCE`：跳跃力度
- `PLATFORM_WIDTH`：平台宽度
- `BASE_DISTANCE`：基础跳跃距离

### 视觉效果调整
- 背景元素数量：修改各渲染函数中的循环次数
- 动画速度：调整 `time` 乘数
- 元素大小：修改 `scale` 参数
